# Build

```bash
# install the dependencies
npm install
# build the code
npm run build
# run all the tests
npm run test
```

# Run

```
npm start day01
```

# Build + Watch

```bash
# The constantly build the code
npm run watch
# The constantly run tests
npm run test-watch
```
